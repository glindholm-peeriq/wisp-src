	Instructions on how to create an ACUCOBOL runtime with EDE
	==========================================================

	1) Copy the library files to /usr/lib.

		Copy each of the following files from the WISP and EDE
		distribution kit to /usr/lib.

		$ cp wisp/lib/libwisp.a   /usr/lib
		$ cp wisp/lib/libvideo.a  /usr/lib
		$ cp wisp/lib/libwtest.a  /usr/lib
		$ cp wisp/lib/libilp.a	  /usr/lib
		$ cp ede/libede.a	  /usr/lib

	2) Copy the .umf file and sub85.c to the acucobol directory

		Note: The (acucobol dir) is the directory where you loaded
		      your distribution from AcuCOBOL.  It should contain
		      the files runcbl.a, sub.c config85.c etc.

		$ cp wisp/acu/sub85.c  (acucobol dir)
		$ cp ede/wruncble.umf  (acucobol dir)
		$ cp ede/wruncbl2e.umf (acucobol dir) - if using 2.0 ACUCOBOL

	3) Build the runtime.

		$ cd (acucobol dir)
		$ make -f wruncble.umf     - or wruncbl2e.umf

	4) Setup to run

		- copy wruncble (or wruncbl2e) to your bin directory
		- edit $WISPCONFIG/wruncbl and set RUNCBL=wruncble or 2e
