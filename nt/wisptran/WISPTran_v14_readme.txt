WISPTran v1.4 Readme
====================
This release of WISPTran was suppose to be part of the WISP 4.4.00 release
but was missed.

This version contains the following fixes and changes:

- On Windows 2000 (and XP) the output was being displayed as all asterisks.

- Mismatched background colors have been fixed.

- Add the WISP translation option /X and /M.

- Fixed problem with output being trunctated.

- Fixed menu problem where "Print" and "Clear" options were not active.

- Fixed display problem with timestamp.
